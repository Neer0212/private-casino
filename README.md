# 🎰 Private Node.js Casino

A real-time, multiplayer virtual casino built with Node.js, Express, Socket.IO, and SQLite. 

## Features
* **Real-Time Multiplayer:** Play Blackjack alongside your friends at the same virtual table.
* **Persistent Economy:** SQLite database automatically saves player balances across sessions.
* **Smart Dealer AI:** Server-side validation handles complex Blackjack math, dynamic Aces, and hardcoded dealer rules.

## How to Run Locally
1. Clone the repo: `git clone [YOUR-REPO-LINK-HERE]`
2. Install dependencies: `npm install`
3. Start the server: `node server.js`
4. Open `http://localhost:3000` in your browser.
